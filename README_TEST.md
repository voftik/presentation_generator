# Тестер качества презентации

## Описание

`test_presentation.py` - скрипт для проверки качества созданной PowerPoint презентации.

## Что проверяется

### 🖼️ Изображения
- ✅ Наличие изображений на всех слайдах
- ✅ Консистентность геометрии (позиции и размеры)
- ✅ Одинаковое количество изображений на каждом слайде

### 📐 Геометрия элементов
- ✅ Консистентность позиций заголовков (по типам макетов)
- ✅ Консистентность позиций контентных блоков
- ✅ Правильное расположение элементов

### 🔤 Шрифты
- ✅ Использование правильного шрифта (Montserrat)
- ✅ Правильные размеры шрифтов:
  - Section Header заголовки: 36pt
  - Content заголовки: 30pt  
  - Контент: 18pt

### 📏 Границы текста
- ✅ Текст не выходит за границы слайда
- ✅ Нет переполнения контентом
- ⚠️ Предупреждения о слишком длинных строках

### 🎨 Макеты слайдов
- ✅ Правильное использование макетов
- ✅ Section Header для H1 заголовков
- ✅ Title and Content для H2 заголовков

## Примеры проблем

### Критические проблемы (❌)
- Отсутствие изображений
- Текст выходит за границы слайда
- Неправильные шрифты
- Несовместимые позиции элементов

### Предупреждения (⚠️)
- Слишком длинные строки текста
- Неожиданные макеты слайдов
- Небольшие отклонения в геометрии

## Интеграция в workflow

Рекомендуется запускать тест после каждого изменения генератора:

```bash
python main.py && python test_presentation.py
```
