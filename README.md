# 🎨 AI-Powered PowerPoint Generator

Интеллектуальный генератор презентаций PowerPoint с использованием Claude AI для создания профессиональных презентаций из текстового контента.

## ✨ Возможности

### 🤖 Многослойная AI-обработка
- **Оптимизация контента** - Claude AI анализирует и сокращает объемный контент до оптимального размера
- **Экспертное улучшение** - Специализированные промпты для создания контента уровня руководителей
- **Второй слой улучшений** - Оптимизация заголовков, текста и визуального оформления

### 🎯 Автоматизированный пайплайн
1. **Оптимизация** → Сокращение 300+ слайдов до ~60
2. **Валидация** → Проверка структуры и качества
3. **Улучшение** → Экспертный анализ и доработка 20-25 ключевых слайдов
4. **Второй слой** → Финальная оптимизация заголовков и макета
5. **Тестирование** → Автоматическая оценка качества

### 🎨 Профессиональное оформление
- **Адаптивный дизайн** - Чередующееся выравнивание текстовых блоков
- **Типографика** - Жирные заголовки, оптимальный размер шрифта (20pt)
- **Макет** - Уменьшение ширины текста на 40%, улучшенное позиционирование
- **Изображения** - Зеркальное отражение для сбалансированной композиции

### 📊 Умное форматирование
- **Заголовки** - 5-6 слов, разбитие на 2 строки
- **Контент** - Лаконичные абзацы, профессиональный стиль
- **Поддержка Markdown** - Автоматическое форматирование **жирного** текста

## 🚀 Быстрый старт

### Установка

```bash
git clone https://github.com/voftik/presentation_generator.git
cd presentation_generator
python3 -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### Настройка

1. Получите API ключ Claude от Anthropic
2. Установите переменную окружения:
```bash
export ANTHROPIC_API_KEY="your-api-key-here"
```

3. Поместите ваш контент в `content/content.md`
4. Убедитесь, что шаблон находится в `tempate/Шаблон презентации 16х9.pptx`

### Запуск

```bash
# Полный автоматический пайплайн
python smart_generator.py

# Или отдельные модули
python content_optimizer.py      # Только оптимизация
python presentation_enhancer.py  # Только улучшение
python main.py                   # Только генерация
```

## 🔧 Модули

### SmartPresentationGenerator
Главный оркестратор, управляющий полным циклом создания презентации.

```python
from smart_generator import SmartPresentationGenerator

generator = SmartPresentationGenerator()
success = generator.generate_smart_presentation()
```

### ContentOptimizer
AI-модуль для оптимизации большого объема контента.

```python
from content_optimizer import ContentOptimizer

optimizer = ContentOptimizer(api_key, model)
optimizer.optimize("input.md", "output.md")
```

### PresentationEnhancer
Экспертное улучшение ключевых слайдов с помощью Claude AI.

```python
from presentation_enhancer import PresentationEnhancer

enhancer = PresentationEnhancer(api_key, model)
enhancer.enhance_presentation("input.md", "enhanced.md")
```

### PresentationEnhancerLayer2
Финальная оптимизация заголовков и визуального оформления.

```python
from presentation_enhancer_layer2 import PresentationEnhancerLayer2

layer2 = PresentationEnhancerLayer2(api_key)
layer2.enhance_layer2("input.pptx", "output.pptx")
```

## 🎯 Особенности

### AI-Оптимизация
- **Claude Sonnet 4** - Самая продвинутая модель для анализа контента
- **Температура 0.9** - Высокая креативность для экспертного контента
- **Контекстная обработка** - До 20,000 токенов за запрос
- **Таймауты** - 300 секунд для сложных задач

### Профессиональный контент
- **Целевая аудитория** - Руководители и эксперты Гознака
- **Стиль** - Деловой, убедительный, структурированный
- **Качество** - Экспертный уровень анализа и рекомендаций

### Визуальный дизайн
- **Чередующееся выравнивание** - Текст слева/справа через слайд
- **Динамичный макет** - Изображения зеркально отражаются
- **Типографика** - Жирные заголовки, 20pt основной текст
- **Композиция** - Оптимальное позиционирование элементов

## 📊 Форматы контента

### Входной формат (Markdown)
```markdown
# Основной заголовок презентации

## Раздел 1

### Слайд 1
Содержимое слайда с **жирным** текстом.

### Слайд 2
Еще один слайд с важной информацией.
```

### Выходной формат
- **PowerPoint (.pptx)** - Полностью отформатированная презентация
- **Профессиональный дизайн** - Готово для представления
- **Мультимедиа** - Поддержка изображений и графики

## 🔍 Тестирование

Автоматическое тестирование качества включает:
